# SystemControl++

A basic yet ambitious C++ project that simulates a modular system manager.

## Modules

- File Manager (create/delete files in `data/`)
- Process Manager (simulated processes)
- System Monitor (simulated CPU/RAM stats)
- User Manager (basic user records)
- Tools (calculator, password generator, simple encryption)

## Build & Run

```bash
g++ src/*.cpp -I include -o SystemControlPP
./SystemControlPP
